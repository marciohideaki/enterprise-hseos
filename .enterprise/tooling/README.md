# Enterprise Tooling
**Scope:** Governance-supporting scripts  
**Execution:** Manual only  
**Status:** Controlled  

---

## Purpose
This directory contains **auxiliary scripts** that support Enterprise governance.

These scripts:
- Do NOT run automatically
- Do NOT replace human judgment
- Do NOT modify code unless explicitly stated

They exist to:
- Guide disciplined processes
- Reduce human error
- Enforce explicit contracts when required

---

## Execution Rules
- Scripts MUST be run manually
- Scripts MUST be read before execution
- Scripts MUST NOT be assumed safe outside their scope

Automation without intent is forbidden.

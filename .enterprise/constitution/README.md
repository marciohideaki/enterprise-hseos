# Enterprise Constitution

This directory contains the **constitutional documents** of the Enterprise Overlay.

The constitution defines the **highest-level principles** governing:
- authority
- decision-making
- separation of responsibilities
- mandatory behaviors for humans and agents

---

## What Lives Here
- Core constitutional documents
- Non-negotiable governance principles


---

## What Does NOT Live Here
- Product requirements
- Stack specifications
- Implementation details
- Execution playbooks

---

## Authority
Documents in this directory have **constitutional authority**.

All policies, playbooks, agents, and tooling MUST comply with these principles.

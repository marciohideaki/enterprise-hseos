# Commit Hygiene — Quick Check

> Tier 1: use for every commit, PR title, and changelog generation.
> Load SKILL.md (Tier 2) only if unclear how to fix a failing item.

---

## Checklist (ALL must pass)

- [ ] No co-authorship trailers: `Co-authored-by`, `Co-authors`, or any trailer crediting a tool or assistant
- [ ] No AI tool names (whole-word match — see matching rules below)
- [ ] Subject follows format: `<type>(<scope>): <imperative summary>`
- [ ] Message describes **what changed and why** — not the process or tool used

---

## Blocked Terms (Whole-Word Match Only)

Match as **complete words** using word boundaries (`\b...\b`).
**Never match substrings.** A word that merely contains a blocked sequence is NOT a violation.

| Term | Case | Blocked | NOT blocked (examples) |
|---|---|---|---|
| `AI` | case-insensitive whole-word | "an AI system", "AI-assisted" | **domain**, **email**, **detail**, **maintain**, **trail**, **curtail**, **retail**, **email**, **claim**, **air** |
| `HSEOS` | exact | "HSEOS", "using HSEOS" | — |
| `Claude` | exact | "Claude generated" | — |
| `Gemini` | exact | "Gemini API" | — |
| `Codex` | exact | "via Codex" | — |
| `Copilot` | exact | "GitHub Copilot" | — |
| `Cursor` | exact | "Cursor IDE" | `cursor` (DB cursor, UI cursor) |
| `ChatGPT` | exact | "ChatGPT output" | — |
| `OpenAI` | exact | "OpenAI model" | — |
| `Anthropic` | exact | "Anthropic Claude" | — |
| `LLM` | exact | "LLM response" | — |
| `AI-assisted` | phrase | always | — |
| `generated by AI` | phrase | always | — |
| `assisted by AI` | phrase | always | — |

> ⚠️ `agent` and `prompt` are **not automatically blocked** — they are common technical terms
> (user-agent, agent pool, CLI prompt, prompt user for input).
> Block only when they explicitly refer to AI tooling in context.

---

## False Positive Guard — NEVER flag these

Words that **contain** a blocked substring but are NOT violations:

```
domain, email, detail, maintain, trail, curtail, retail, claim
AirBnB, airport, air (context: air conditioning)
maintain, container, entail, attain, certain, captain
raid, paid, afraid, maid, said, laid
```

If your scan flagged one of these → **PASS**. Do not rewrite.

---

## Verdict

**PASS** → all items clear, proceed.
**FAIL** → rewrite the commit message: remove AI/tool attribution, keep only the technical change summary.

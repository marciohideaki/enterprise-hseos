# Enterprise Modes

This directory defines **explicit operational modes**.

Modes alter governance behavior temporarily and explicitly.

---

## What Lives Here
- Replay Mode definition
- Mode activation artifacts
- Mode closure records

---

## Key Rule
Modes:
- MUST be explicitly activated
- MUST be explicitly deactivated
- MUST have a clear closure

If a mode is active, its rules are mandatory.

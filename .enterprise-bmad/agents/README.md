# BMAD Enterprise Agents

This directory defines the **roles, authority, and constraints** of all BMAD agents.

Agents are **executors**, not decision-makers.

---

## What Lives Here
- Authority definitions per agent
- Explicit constraints per agent
- Scope boundaries

---

## What Agents CANNOT Do
- Redefine stack or architecture
- Override specifications
- Resolve conflicts autonomously
- Bypass ADR requirements

---

## Important Principle
Agents operate under **strict authority limits**.

If ambiguity or conflict is detected:
- Execution MUST stop
- ADR MUST be requested

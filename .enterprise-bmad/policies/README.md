# BMAD Enterprise Policies

This directory contains **operational governance policies**.

Policies define:
- WHEN to act
- WHEN to stop
- WHEN escalation is mandatory
- WHEN ADRs are required

---

## What Lives Here
- ADR policies
- Documentation policies
- Validation policies
- Sharding rules
- Exception handling rules

---

## What Policies Are NOT
- Not implementation guides
- Not architectural specs
- Not best-practice suggestions

Policies are **enforceable rules**, not recommendations.

---

## Relationship to Constitution
Policies MUST comply with the BMAD Enterprise Constitution.

If a policy conflicts with the constitution, the policy is invalid.

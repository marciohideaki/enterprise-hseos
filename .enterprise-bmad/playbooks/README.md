# BMAD Enterprise Playbooks

This directory contains **human-readable operational guides**.

Playbooks explain HOW to execute workflows that are already defined
by the constitution and policies.

---

## What Lives Here
- Onboarding flows
- Escalation procedures
- Replay workflows
- Repository curation guides

---

## What Playbooks Are NOT
- Not policies
- Not sources of authority
- Not substitutes for specifications

Playbooks provide guidance, not permission.
